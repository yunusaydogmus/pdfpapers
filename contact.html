Questions fréquentes

Comment puis-je annuler mon abonnement ?
Vous pouvez annuler votre abonnement à tout moment depuis votre tableau de bord, section "Mon compte → Abonnement". L'annulation prend effet à la fin de la période en cours.
Mes fichiers sont-ils vraiment supprimés après traitement ?
Oui, tous vos fichiers sont automatiquement et définitivement supprimés de nos serveurs après 2 heures (plan gratuit) ou 24 heures (plan Pro). Vous pouvez aussi les supprimer manuellement.
Proposez-vous une API pour développeurs ?
Oui, notre API REST est disponible pour les clients Business. Contactez-nous à partners@pdfpapers.com pour la documentation et les clés d'API.
Y a-t-il une application mobile PDFPapers ?
Notre site web est entièrement optimisé pour mobile et fonctionne parfaitement sur iOS et Android. Une application native est en cours de développement.
